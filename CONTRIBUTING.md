# Contributing to OCRC

We welcome contributions from everyone! By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).

## How to contribute

- **Reporting issues**: Use GitHub Issues to report bugs, suggest enhancements, or ask questions.
- **Pull requests**: 
  1. Fork the repository.
  2. Create a branch for your changes (`git checkout -b feature/your-feature`).
  3. Commit with clear messages.
  4. Open a pull request describing your changes.
- **Discussions**: Join our online meetings (see [community/events](community/events/)) or start a discussion in GitHub Discussions.

## Guidelines

- Keep files minimal and self-contained.
- Use clear, descriptive names.
- Document any new fields or formats.
- Ensure backward compatibility when extending schemas.

Thank you for helping make OCRC a valuable resource for all researchers!

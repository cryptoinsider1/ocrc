# Open Cooperative Research Protocol - OCRC

**Voluntary framework standard for open, interdisciplinary and environmental science.**

This repository contains specifications, templates and tools for using the OCRC protocol. We invite researchers from all fields to join the discussion and development.

## Main components

- [Manifesto](docs/manifesto.pdf) — conceptual description of the protocol.

- [ROI Specification](spec/roi/) — identifiers of research objects.

- [RO-Crate profile](spec/rocrate/) — data packaging.

- [Ontology OCRC Core](spec/ontology/) — general dictionary of concepts.

- [HDF](spec/hdf/) — description of hypotheses.

- [API](spec/api/) — a single query interface.

- [Templates](templates/) - ready-made blanks for a quick start.

- [Examples](examples/) — full examples of use.

## Quick start

1. Read the [manifesto](docs/manifesto.pdf).

2. Select the section you are interested in in [spec](spec).

3. Use templates from [templates](templates) to create your own objects.

4. Share results via RO-Crate and specify ROI.

# Templates

This directory contains reusable templates for creating OCRC‑compliant objects.

- [rocrate_template](https://github.com/cryptoinsider1/ocrc/tree/main/templates/rocrate_template) – basic RO-Crate structure with placeholder metadata.
- `hypothesis_template.json.template` – HDF hypothesis template.
- `event_template.json.template` – FRB event metadata template.
- `source_template.json.template` – FRB source metadata template.
- `notebook_template.ipynb` – Jupyter notebook template with ROI header.

Fill in the `{{VARIABLES}}` and use the files as a starting point.

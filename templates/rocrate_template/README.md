# RO-Crate Template

Copy this directory and replace `{{VARIABLES}}` in [ro-crate-metadata.json.template](https://github.com/cryptoinsider1/ocrc/blob/main/templates/rocrate_template/ro-crate-metadata.json.template) with actual values. Place your data files in `data/`.

Then rename the template file to `ro-crate-metadata.json`.

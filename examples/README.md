# Examples

This directory contains complete examples of OCRC usage.

- [frb180916](https://github.com/cryptoinsider1/ocrc/tree/main/examples/frb180916) – data and hypotheses for FRB 180916.J0158+65.
- [multidisciplinary](https://github.com/cryptoinsider1/ocrc/tree/main/examples/multidisciplinary) – a fictional example combining astronomy and economics.

Each subdirectory is self‑contained and includes RO‑Crates and HDF files.

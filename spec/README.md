# Specifications

This directory contains the technical specifications for each component of the OCRC protocol.

- [roi/](roi/) – Research Object Identifier format.
- [rocrate/](rocrate/) – OCRC profile for RO-Crate.
- [ontology/](ontology/) – OCRC Core Ontology.
- [hdf/](hdf/) – Hypothesis Description Framework.
- [api/](api/) – GraphQL API schema.

Each subdirectory includes a README, specification document, examples, and relevant schemas.

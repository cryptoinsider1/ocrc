# Tools

Command‑line tools for working with OCRC.

- [roi-generator](https://github.com/cryptoinsider1/ocrc/tree/main/tools/roi-generator) – generate ROIs.
- [hdf-validator](https://github.com/cryptoinsider1/ocrc/tree/main/tools/hdf-validator) – validate HDF JSON files.
- [rocrate-validator](https://github.com/cryptoinsider1/ocrc/tree/main/tools/rocrate-validator) – check RO-Crate compliance.

Each tool has its own README and requirements.
